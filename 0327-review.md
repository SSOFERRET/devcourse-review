# :fire: 03/27 배운 내용 정리

## :one: DBMS

#### 데이터베이스란

데이터를 효율적으로 관리하기 위하여 묶은 데이터의 집합체.

#### DBMS

Database Management System

데이터베이스를 관리하는 시스템. SQL 언어를 사용한다.

- Elasticsearch
- Oracle

## :two: RDBMS를 사용하는 이유

Relational DBMS

데이터 간의 관계를 토대로 데이터에 접근 할 수 있는 DBMS.

(예: 마인드맵, 배우 공유에 대한 정보에 접근할 때 도깨비 검색을 통함.)
